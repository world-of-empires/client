## Когда обновлять

| Событие | ARCHITECTURE.md | AI_CONTEXT.md |

| Новый слой/папка | ✅ | — |
| Изменение правил именования | ✅ | — |

| Новый/изменённый/удалённый экспорт | — | ✅ |
