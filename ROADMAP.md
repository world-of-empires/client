# World of Empires — Roadmap

Дорожная карта разработки клиентской части игры.  
Отмеченные пункты считаются завершёнными на текущем этапе.

**Текущая версия:** v0.2.2 | **Последнее обновление:** 12.02.2025

---

## Фаза 0: Инициализация проекта — v0.1.0 (12.02.2025)

- [x] Создание Next.js приложения
- [x] Подключение TypeScript
- [x] Подключение Tailwind CSS v4
- [x] Настройка ESLint
- [x] Настройка Prettier
- [x] Настройка path alias (`@/*`)
- [x] Добавление Pixi.js
- [x] Базовый layout и шрифты (Geist)

## Фаза 2: Игровой движок (Pixi.js) — v0.1.0 (12.02.2025)

- [x] Инициализация Pixi Application

- [x] Динамический импорт canvas (SSR отключён)

- [x] Game loop (ticker)
- [x] Демо-объект (вращающийся квадрат)

## Фаза 3: UI и HUD — v0.1.0 / v0.2.0 (12.02.2025)

- [x] HUD поверх canvas (position: fixed, z-index)
- [x] Кнопка «Выход» в HUD — удалена, навигация через браузер
- [x] Заглушка «Счёт» — заменена на Legend, HoverInfo, Controls
- [x] Controls: кнопка «Новая карта», отображение seed
- [x] Legend: легенда биомов с цветами
- [x] HoverInfo: подсказка при наведении (биом, координаты)
- [x] Hint: подсказка по управлению (Drag, Scroll)
- [ ] Интеграция реального счёта
- [ ] Дополнительные элементы HUD (ресурсы, меню и т.п.)

## Фаза 4: Ассеты и тайлы — v0.2.0 (12.02.2025)
