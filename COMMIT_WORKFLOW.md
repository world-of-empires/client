## Чек-лист

1. Код работает, lint, tsc
2. AI_CONTEXT.md (если новый/изменённый API)
3. CHANGELOG.md — новая запись
4. package.json — версия
5. AI_CONTEXT.md, ROADMAP.md — версия
6. `git add -A` → `git commit`
